# FCC Python — Lab & Workshop Solutions

Solutions to the labs and workshops from the [freeCodeCamp Python Certification](https://www.freecodecamp.org/learn/python-v9) (Part of the Full Stack Developer v10 curriculum).

---

## About

This repository contains my personal solutions to the hands-on labs and workshops completed throughout the certification. Each file is a standalone Python script targeting a specific concept or problem set covered in the curriculum.

> **Note:** These are individual exercise solutions, not the five certification projects required to unlock the final exam.

---

## Files

| File | Description | Concepts |
|------|-------------|----------|
| [`Pin_extractor.py`](./Pin_extractor.py) | Extracts secret PIN codes hidden in poems by reading word counts diagonally across lines | Loops, string methods, list indexing |
| [`email_sim.py`](./email_sim.py) | Simulates email inbox operations | Functions, strings, data manipulation |
| [`game_char_stats.py`](./game_char_stats.py) | Manages and queries game character statistics | Dictionaries, data modeling |
| [`medical_data_valid.py`](./medical_data_valid.py) | Validates and processes medical records | Error handling, data validation |
| [`musical_inst_inv.py`](./musical_inst_inv.py) | Manages a musical instrument inventory system | Dictionaries, sets |
| [`number_pattern_gen.py`](./number_pattern_gen.py) | Generates number patterns from input sequences | Loops, sequences, logic |
| [`planet_class.py`](./planet_class.py) | Models planets using OOP principles | Classes, objects, attributes |
| [`salary_tracker.py`](./salary_tracker.py) | Tracks and calculates employee salary data | Dictionaries, arithmetic, aggregation |
